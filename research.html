Research Topics

研究理念：以研究安全领域实际问题为导向，以创新方法/模型构建培养学生研究能力/兴趣！
研究机会：每年大创、互联网+、信息安全作品赛、毕业设计有部分名额供本科生选择，请提前联系！

Web/network attack detection and forensics

This topic focus on introducing machine learning and NLP technologies to solve cyber security issues, especially for network attack dectection, behavior anomaly detection, backdoor analysise, honeypot construction and idenfiticaiton methodology and etc.

  • Web应用攻击(SQL/XSS/Phishing/Mail)检测
  • Webshell/Powershell/JavaScript恶意文件检测
  • DNS匿名通信/DGA域名/域名Parking检测
  • Web应用/系统/蜜罐日志异常挖掘
  • 其他相关内容

Source code and supply-chain security

This topic focus on automatically finding web vulnerability or malicious codes in public code repositories or package repository of softwares, especially for Python/PHP/Ruby/Java.

  • Web应用漏洞智能化挖掘方法
  • 恶意库自动化识别与分析
  • 开源组件中的漏洞挖掘
  • 浏览器插件安全性分析
  • 程序员社区中的代码安全
  • 其他相关内容

Cybercrime and threat intelligence analyze

This topic usually collects public social data from the Internet, then proposes different models based on machine learning and knowdge graph algorithms to solve these things: social user spam detction, fake news and fraud behavior detection, cybercrime activity or key authors portrait, automatically extract threat intelligence (IP, domain, technique) and etc.

  • 社区内容中恶意域名智能识别
  • 网络社区中安全新词提取
  • 非结构化数据中的威胁情报图谱构建
  • 虚假消息/用户的识别问题
  • 社区中人员关系及影响力分析
  • 其他相关内容
Artificial Intelligence in Medicine (Interdisciplinary)

The topic usually introduces machine learning techniques, and data streaming algorithms to improve the current important problems in medicine. Digging the inner relations between different report factors in medicine and prediction the possiblility of cure.

  • XX病的临床因素关联挖掘
  • 药物分子结构的毒性智能关联
  • 临床手术的成功机率及相关因素
  • 其他相关内容
